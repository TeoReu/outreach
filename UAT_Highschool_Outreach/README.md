# High School Outreach — UAT

This folder contains the resources used for the Universal Approximation Theorem (UAT) high school outreach session.

## Included resources

- **Presentation:** `Balliol Outreach - UAT.pdf`
- **Jupyter notebook:** `StudentPerformanceMLP.ipynb`

## About this project

This project introduces high school students to neural networks and multilayer perceptrons (MLPs) through a practical notebook example. It includes a hands-on notebook for student performance prediction and a presentation to support the workshop delivery.

## Recommended learning resources

These external resources are great for getting more depth on neural networks, deep learning, and interactive experimentation:

- [Neural Networks and Deep Learning — Chapter 4](http://neuralnetworksanddeeplearning.com/chap4.html)
- [Neuromatch Academy — MLP tutorial](https://deeplearning.neuromatch.io/tutorials/W1D3_MultiLayerPerceptrons/student/W1D3_Tutorial1.html)
- [Neural approximation 1D interactive demo](https://12345pnp.github.io/comp-play/neural/networks/2022/11/13/neural-approx-1D.html)
- [TensorFlow Playground](https://playground.tensorflow.org/)

## How to use

1. Open `Balliol Outreach - UAT.pdf` for the presentation slides.
2. Open `StudentPerformanceMLP.ipynb` in Jupyter Notebook or JupyterLab.
3. Review the notebook cells and run them in order to explore how a multilayer perceptron works.
4. Use the links above for additional tutorials and interactive neural network visualizations.

## Notes

- The notebook is intended for educational use and demonstration.
- The included presentation supports the main concepts and examples used in the session.
